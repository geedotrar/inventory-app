## Requirement

- PHP 8.3+
- Composer

## How to Run

1. Install dependency PHP.

   ```bash
   composer install
   ```

2. Jalankan migrasi dan seeder.

   ```bash
   php artisan migrate --seed
   ```

3. Jalankan server Laravel.

   ```bash
   php artisan serve
   ```
