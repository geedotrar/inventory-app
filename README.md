
# inventory-app

## Requirement

- PHP 8.3+
- Node.js 20.19+ atau 22.12+
- Composer

## How to Run

1. Install dependency PHP.

   ```bash
   composer install
   ```

2. Jalankan migrasi dan seeder.

   ```bash
   php artisan migrate --seed
   ```

3. Jalankan server Laravel.

   ```bash
   php artisan serve
   ```

## Testing

```bash
php artisan test
```
